2021 普通电脑挖矿入坑超详教程

https://www.bilibili.com/read/cv10962997

最近虚拟货币火热，许多人想尝试挖矿来获得虚拟货币，但很多人没有矿机矿卡，用一台普通的家用电脑怎么挖矿呢？用 CPU

挖比特币是不可能的，但是我们有这样一个虚拟货币――门罗币（Monero，XMR）。它的 RandomX 算法对 cpu 优化极高，抵制矿机，使 CPU 挖矿成为可能。（文章撰写时 xmr 价格 2316￥/个）那么，如何开始呢？

一：下载挖矿软件（xmrGuiMiner）

百度网盘 https://pan.baidu.com/s/1EdN0ZVUCAri7TK2mGSytMw 提取码: w33n

GitHub：https://github.com/EntropyRiser/xmrGuiMiner/releases/download/v1.0.2/xmrGuiMiner.zip

（GitHub 上最可以检查新版本 https://github.com/EntropyRiser/xmrGuiMiner/releases）

下载后解压，打开 xmrGuiMiner.exe（需要.NET Framework4)，界面如图所示。

 

获取 XMR 钱包地址想要保存门罗币，我们需要一个钱包，这里推荐 mymonero.com。

点          击         “         关         于          ”         ，         打         开         在         线             钱        包        链         接        。

 

 

创建一个新钱包，这里有一个口令，要保存下来，不然就无法登录了//

 点击 copy 复制你的收款地址~

 回      到      xmrGuiMiner ，      填      写      你      的      收      款      地      址      ，          点      击      开      始      挖      矿      ，      duang~  散热不好就直接起飞了！调节 cpu 选项下面的滑块可以调整 cpu 占用量（需要重启挖矿软件）这个时候我们点击“关于”下的“矿池主页”，来到矿池官网，输入付款地址，就可以看到收益了！达到 0.01xmr 就可以选择提取了。
